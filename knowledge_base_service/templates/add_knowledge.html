4️⃣ 文件存放路径

批量导入的文件可以通过以下方式处理：

方式一：Web界面上传

在"批量导入"标签页中直接拖拽或选择文件上传，系统会自动处理。

方式二：手动放置文件

将准备好的文件放置到以下目录：

knowledge_base_service/
├── import_data/           # 创建此目录用于存放导入文件
│   ├── products.json      # 产品数据文件
│   ├── knowledge.json     # 知识数据文件
│   └── articles.md        # Markdown知识文件
└── layered_knowledge_base/ # 知识存储目录（系统自动管理）
    ├── domain_background/
    ├── specific/
    └── associative/

然后调用API接口处理：POST /api/v1/knowledge/import

5️⃣ 字段说明

产品信息字段：

  • name (必填) - 产品名称
  • brand - 品牌
  • reference_price - 参考价（数字）
  • category - 产品分类
  • efficacy - 主要功效
  • applicable_skin - 适用肤质
  • description (必填) - 产品描述
  • tags - 标签数组或逗号分隔的字符串

专业知识字段：

  • title (必填) - 知识标题
  • topic - 主题分类
  • layer - 知识层级（specific/domain_background/associative）
  • content (必填) - 知识内容
  • source_url - 来源URL
  • tags - 标签数组或逗号分隔的字符串
